- Core feature 3: Add the wathced movies to a watched list and give it a rating

## Getting Started

### Prerequisites
- Node.js 18+
- Java 17+
- PostgreSQL

### Installation
1. Clone the repo
   ```bash
   git clone https://github.com/ParseDotEXE/fullstack-movie-watchlist-app.git
   ```
2. Install dependencies
   ```bash
   npm install
   ```
3. Set up environment variables
   ```bash
   cp .env.example .env
   # Add your API keys
   ```
4. Run the application
   ```bash
   npm start
   ```
